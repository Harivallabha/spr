# abstract-world-models

### Dependencies: 
* PyTorch 
* wandb (logging tool)
* gym

To install the requirements, follow these steps:
```bash
# PyTorch
conda install pytorch torchvision -c pytorch

# Other requirements
pip install -r requirements.txt
```
